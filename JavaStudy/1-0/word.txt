IT　Information Technology、情報技術のこと

ICT　Information and Communication Technologyの略称、情報通信技術を利用したコミュニケーションのこと

IoT　Internet of Thingsの略称、ものに通信機能を搭載してインターネットに接続、連携させる技術

OS　Operating System、コンピュータを動かすのに使うソフトのこと

Windows　マイクロソフトが開発、販売するOSの名称

macOS(Mac OS X)　アップル社が開発、販売するOSの名称

UNIX　現存する中で最も古いOSのひとつ

Linux　無料で配布されている使用率の高いOS

HTML　Hyper Text Markup Languageの略称、Webページを作るための最も基本的な言語のひとつ

CSS　Cascading Style Sheetsの略称、HTMLは文書構造を作るためのものに対し、こちらは文書構造の見栄えを整えるもの

JavaScript　Webページの開発などに使われるスクリプト言語、どんなコードで書かれているかなどが閲覧者が見ることができる

.NET(C#,VB)
・C#　標準言語のC#を使ってMicrosoftが開発したコンパイラのこと
・VB　Microsoftが開発した言語、初めてでも学びやすい

Java　プログラミング言語のひとつで、汎用性も高く、多く使用される。

Java SE　Java Standard Editionの略称、Javaの基本機能をまとめたもの

Java EE(Jakarta EE)　Java Platform, Enterprise Editionの略称、Javaで企業向けのWebアプリケーションを開発するときに必要な機能
・Jakarta EE　Java EEの後継として存在する。多くのAPIを統合したもの

Perl　Practical Extraction and Report Languageの略称、開発言語のひとつ。古くから存在し自由度の高いもの

PHP　Webページの開発などに使われるスクリプト言語、どんなコードで書かれているかなどは閲覧者が確認できない

Ruby　日本の技術者が開発したプログラミング言語、わかりやすく自由度も高い

Apache　Webサーバのソフト、仕様率が高いもののひとつ

nginx　オープンソースのWebサーバソフトウェアの一つ、同時アクセスが増えても安定的に動作する

IIS　Internet Information Servicesの略称、無料で利用できるWebサーバーソフトでシェアはApacheよりも高い

SQL　Structured Query Languageの略称、データベース管理の際に使用する言語でプログラミング言語ではない

DB2　IBM社が提供しているデータベース管理システム、または関連ソフトウェアのブランド名

MySQL　使用率の高い、データベース管理システム

Maria DB　MySQLから派生したデータベース管理システム、拡張性や処理機能が高く汎用性の高いデータベース

Oracle DB　Oracle社が提供している商用データベースで、世界的にも高いシェアを占めている

PostgreSQL　オープンソースのデータベース、豊富な機能と高い拡張性が評価されている

SQLite　オープンソースのデータベース、ほかのソフトウェアに組み込むことが想定されているため軽量

SQL Server　Microsoftが提供しているデータベース管理システム、高機能なシステムから小規模のシステムまで幅広く対応する

CRUD　データ上の操作に欠かせない「Create」「Read」「Update」「Delete」の頭文字を組み合わせたもの

トランザクション　ソフトウェアの処理方法のひとつで、互いに関連している複数の処理をまとめ、一つの処理として扱うこと

コミット　処理や変更を確定、反映させるという意味

ロールバック　システム障害などでデータの損失、破損が起きた際に、以前に正常に作動していた時点の状態に戻し、復旧を試みることを指す

排他制御(排他ロック)　データやファイルを一人が利用している場合に、整合性を保つためほかの人からのアクセスを制限、または禁止(排他ロック)すること

楽観ロック　ロックは行わず、更新対象のデータが、取得時と同じであれば更新を行うという方法

悲観ロック　データ取得時にロックをかけ、更新が終わってからロックを解く方法

実行計画　より短い時間でSQL文を実行できるか、計算して導き出された、具体的な方法をまとめたもの

チューニング　システムやソフトウェアなどの設定や構成を調整して、目的の状態に近づけたり、性能を最大限引き出したりする作業を指す

スロークエリ　データベースにおいて、時間のかかっているSQL(データベースを管理、操作する言語)を指す

クエリログ　実行されたSQL文がどんなものだったか記録されているもの

IDE(統合開発環境)　Integrated Development Environmentの略称、ソフトウェア開発に必要なソフトウェアを一つに組み合わせ、開発をしやすくしたもの

Eclipse　様々なプログラミング言語によるソフトウェア開発に対応した、IDEのひとつ

NetBeans　プロジェクト管理機能や入力支援機能を備えたオープンソースのIDE

SpringToolSuite　Eclipseをベースとして作成されたIDE、Spring Frameworkと呼ばれるフレームワークを利用した開発に適した機能が備わっている

DBツール　データベース上のデータを扱うために使用するツールのこと

A5:SQL Mk-2　データベース開発において生産性を向上させるべく開発されたSQL開発ツール　

pgAdmin　GUIで行うPostgreSQLの管理ツール

DBeaver　Oracle,MySQLなど42種類に対応したSQLクライアントツール

MySQL Workbench　MySQLのサイトで配布されている、データベースの設計・開発・管理をひとつでできるツール

Sequel Pro　データベースをExcelのような表で確認・編集できるMac専用のアプリ　

TeamSQL　DBをGUIで管理するためのソフトウェアのひとつ　

PM　プロジェクトマネージャーのこと。運営、品質、納期などプロジェクトを円滑に進める役割を果たす管理者のこと

PL　プロジェクトリーダーのこと。システムの構築・導入においてプロジェクトを管理・統括する業務

PG(PE)　プログラマのこと。システムのプログラムを作る人で、プログラム言語を用いてコードを書く

SE　システムエンジニアのこと。顧客の要求から仕様を決定し、大まかな設計をするまでの工程を担当する

デザイナー　製品やポスターのレイアウトなど、様々なもののデザインを行う仕事

エンジニア　工学に関する専門的な知識やスキルを持った人材を指す言葉

フロントエンド・エンジニア　エンジニアの中でも、利用者が直接操作したりする部分を担当する職種

サーバーサイド・エンジニア　サーバー側で動作するプログラムの開発やデータ処理などをおこなうエンジニアのことを指す

バックエンド・エンジニア　プログラムの開発からWebサーバー、データベースといった環境整備まで担当するエンジニアのこと

UI　ユーザーインターフェイスのこと。ユーザーと製品やサービスの接点すべてを指す

CUI　Character User Interfaceの略称、テキストデータのみで示されるUIのこと、コマンドプロンプトなど

GUI　Graphical User Interfaceの略称、CUIに対し、絵や図などで示されるUIのこと、デスクトップ画面など

ファイル　コンピュータにおけるデータの管理単位のひとつで、ストレージなどにデータを記録する際に利用者やOSから見て最小の単位となるデータのまとまり

拡張子　ファイル名の末尾につけられる、種類を識別するための文字列

ディレクトリ　コンピュータの記録媒体上の、すべてのファイルの名前や場所を記録している部分

ルート・ディレクトリ　そのコンピュータ上のディレクトリの中でも最初、または最上位のディレクトリのこと

カレント・ディレクトリ　ディレクトリ上などで作業をしている際、現在作業中のディレクトリのこと

パス　コンピュータ内でのファイルなどの所在を表す文字列のこと

絶対パス　最上位の階層から見て、目的のものがある場所を伝える方法

相対パス　今いる階層や場所を基準にして、目的のものがある場所を伝える方法

LAN　Local Area Networkの略称、限られた範囲内のコンピュータや通信機器などをケーブルや無線機器で接続し、相互にデータ通信できるようにしたネットワークのこと

WAN　地理的に離れた地点間を結ぶ通信ネットワークのこと

クライアント　取引相手や顧客のこと

サーバー　ホームページを表示するために必要な情報を格納している場所のこと

HTTP　Webサーバとクライアント間でデータのやり取りをする通信規約のひとつ(プロトコル)

FTP　サーバとクライアント間でファイルなどの転送をやり取りするプロトコルのひとつ

HTTPS　HTTPによる通信をより安全に行うためのプロトコルのひとつ

SSL　Secure Sockets Layerの略称、送受信しているデータを暗号化する通信手順のひとつ

SSH　Secure Shellの略称、安全に通信を行って、ネットワークに接続された機器を遠隔操作するための通信手段のひとつ

ポート・ポート番号　機器やソフトウェアが外部の別の主体と接続・通信するための末端部分のこと。ポート番号はそれに割り振られた番号。

HTTP、FTP、SSHの各ポート番号　HTTP/80番 FTP/20番と21番 SSH/22番(デフォルトの場合)

ファイアウォール　内部ネットワークをインターネットを通して侵入してくる不正なアクセスから守るための防火壁のこと

TCP　Transmission Control Protocolの略称、インターネットにおいて標準的に利用されているプロトコルのこと

IPアドレス　パソコン、スマートフォン、タブレットなどでネットワーク通信を行うときに利用される機器を判別するのに使われる番号のこと

グローバルIPアドレス　インターネットに接続するときに使用される管理団体に管理されているIPアドレスのこと。

プライベートIPアドレス　組織内のネットワークでのみ使用できるIPアドレスのこと

ドメイン　インターネット上のコンピュータやネットワークを識別するための名前のこと

DNS　Domain Name Systemの略称、インターネット上でドメイン名を管理、運用するために開発されたシステムのこと

VPN　Virtual Private Networkの略称、インターネットなどに接続している利用者の間に仮想的なトンネルを構築し、プライベートなネットワークを構築するシステムのこと

ブリッジ　MACアドレス(PCを識別するためのアドレス)を見て、データの行き先を振り分けしてくれるもの

プロキシサーバー　インターネットに直接接続できないコンピュータの代わりにインターネットに接続し、Webサイトへのアクセスなどを行うサーバーのこと

SMTP　Simple Mail Transfer Protocolの略称、インターネット上で電子メールを伝送するためのプロトコルのひとつ、メールを送信するときに使われる

POP　Post Office Protocolの略称、インターネット上で電子メールを伝送するためのプロトコルのひとつ、メールをクライアント上にダウンロードしてサーバー側から削除するという処理がされる

IMAP　Internet Message Access Protocolの略称、インターネット上で電子メールを伝送するためのプロトコルのひとつ、サーバー側でメールを管理する

セッション　ネットワーク通信などで、接続してから切断するまでの一連の操作や通信のこと

クッキー　ホームページを訪問したユーザーの情報を一時的に保存する仕組み、またはそのデータのこと

キャッシュ　一度アクセスしたサイトやデータを一時的に保管し、次回同じサイトなどにアクセスしたときの表示を早くするもの

インデックス　プログラミングなどで同種のものが複数並んでいるときに、個々の要素を区別するためにつけられる通し番号。検索エンジンでは、格納された大量のデータを素早く検索・抽出するために作成されたと通し番号

フロー　プログラムを構成する命令や手続き、処理の流れなどのこと

タスク　コンピュータが処理する仕事の最小単位のこと

Git　プログラムソースなどの変更履歴を管理するシステム

SVN　ファイルの変更履歴を管理するシステム

デグレ　新しいバージョンのソフトウェアの品質が、以前より悪くなること。デグレードの略称

マージツール　複数のファイルやデータ、プログラムなどを、決められたルールに従ってひとつに統合するために使われるもの
